## 2. FlashAttention-2 演算法與硬體對應

### 2.1 演算法核心（Tiled 計算）

```
輸入：Q[N,d]  K[N,d]  V[N,d]     N=196, d=64, Br=Bc=14
輸出：O[N,d]  (fp32，dequantized)

外迴圈 (r=0..N/Br-1):           ← 14 個 Q-tile
  Q_r = Q[r*Br : (r+1)*Br, :]   ← 從 Host DMA 搬入

  m_r = -inf,  l_r = 0,  O_r = 0

  內迴圈 (c=0..N/Bc-1):          ← 每 Q-tile 掃 14 個 K/V tile
    K_c = K[c*Bc : (c+1)*Bc, :]  ← K Ping-Pong DMA
    V_c = V[c*Bc : (c+1)*Bc, :]  ← V Ping-Pong DMA

    S_rc = Q_r × K_cᵀ  [Br,Bc]  ← 14×14 SA GEMM1 (INT8)
    S_rc = S_rc * scale           ← scale = 1/√d，dequant
    m̃ = rowmax(S_rc)
    P_rc = exp(S_rc - m̃)          ← Online Softmax
    l̃ = rowsum(P_rc)

    O_r = diag(exp(m_r-m̃))⁻¹ * (diag(l_r)*O_r + P_rc × V_c)
                                  ← 14×14 SA GEMM2 (INT8×INT8→fp32)
    更新 m_r, l_r

  O[r*Br:...] = O_r / l_r        ← DMA 寫回 Host
```

### 2.2 Tile 尺寸對應硬體資源

```
┌─────────────────┬──────────┬───────────────────────────┐
│ 參數            │ 數值     │ 硬體對應                  │
├─────────────────┼──────────┼───────────────────────────┤
│ N（patch tokens)│ 196      │ 196 / 14 = 14 個外迴圈    │
│ d（head dim）   │ 64       │ SA 輸出列寬 = 64           │
│ Br = Bc（tile） │ 14       │ SA 尺寸 = 14×14 = 196 PE  │
│ H（heads）      │ 6        │ 6 次獨立 FA call           │
│ L（layers）     │ 12       │ 12 × 6 = 72 次 FA call     │
│ 總 FA cycles    │ 53.8 M   │ 72 × 747,497               │
└─────────────────┴──────────┴───────────────────────────┘
```

## 5. INT8 量化策略

```
┌────────────────────────────────────────────────────────┐
│              Global Symmetric INT8 Quantization         │
│                                                        │
│  scale = max(|X|) / 127                               │
│  X_int8 = clamp(round(X / scale), -127, 127)          │
│                                                        │
│  GEMM1 dequantization:                                 │
│    S_fp = S_int8 × q_scale × k_scale    (independent) │
│    S_fp = S_int8 × qk_scale²            (ScaleTying)  │
│                                                        │
│  GEMM2 dequantization:                                 │
│    O_fp = O_int8 × v_scale                            │
└────────────────────────────────────────────────────────┘

ScaleTying (-DFA_SCALE_TYING):
  qk_scale = max(q_scale, k_scale)
  優點：硬體只需 1 個乘法因子（省 1 次 scale load）
  缺點：當 q_scale ≠ k_scale 時量化精度略降
  實測：對 case2 合成資料 Max diff 維持 1.35e-4（無損失）
```
